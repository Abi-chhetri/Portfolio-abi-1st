PROJECTS

Todo App

A minimal yet powerful task management app designed to help users organize their daily activities efficiently. This project highlights my ability to build interactive and functional user interfaces using React. It demonstrates my understanding of component-based architecture, state handling, event-driven updates, and modern styling techniques. React-js, Tailwind and for statemanagement UseContext was used to create this project.

Features

  • CRUD Operations.
  • Mark tasks as completed with visual feedback.
  • Tasks saved in localStorage (so nothing’s lost on refresh)

Weather App

A sleek weather application that provides real-time weather information based on user location. Designed with an emphasis on clean UI and useful weather insights.

This project showcases my ability to integrate third-party APIs, handle asynchronous data, and build interactive UIs. It also reflects my understanding of working with real-time data, error handling, and geolocation in modern frontend development.

Features

  • Real-time weather data fetched from the MetoWeather API.
  • Auto-fetch weather based on user's location.
  • Displays humidity, wind speed, and weather state with icons.
  • Built with React, Tailwind CSS and CSS3.
